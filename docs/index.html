Knowledge Graph Agnostic Node

Knowledge-Aware LLM Applications

Overview

Knowledge Graph Agnostic Node is a Python library that allows you to retrieve knowledge from any knowledge graph (KG). Simply provide a seed node, and the library traverses the graph to extract relevant information for your LLM applications.

Installation

pip install kg-agnostic-node

Usage Example


from kg_agnostic_node import KGNode

# Initialize with seed node
node = KGNode(seed="Cancer")

# Traverse the knowledge graph
results = node.traverse(depth=3)

print(results)

Features

  • Works with any knowledge graph
  • Simple seed node input
  • Configurable traversal depth
  • Knowledge retrieval optimized for LLMs
